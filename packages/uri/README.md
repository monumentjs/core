# URI

Provides URI manipulation functionality: building, parsing, serializing and validating

Related links:

- [API reference](https://monumentjs.github.io/package/uri)

# Install

Install with NPM:

```
npm i @monument/uri
```

Install with Yarn:

```
yarn add @monument/uri
```
